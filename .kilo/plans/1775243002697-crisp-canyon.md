# UI Refactoring: Refined Filtering and Automated Lifecycle Cleanup

This plan outlines the final logic adjustments to ensure content flows correctly through the pipeline and disappears when published.

## 1. Filter Logic Refinement

### A. Redaktions-Planung (`src/app/planning/editorial-planning.tsx`)
- **Requirement**: Show everything from "Planned" to "Before Published".
- **Logic Change**: Explicitly filter the data provided to the Editorial Planning table to include only:
    - `'Planned'`, `'Beauftragt'`, `'In Arbeit'`, `'Angeliefert'`, `'Review'`, `'Optimierung'`.
- **Exclusion**: `'Published'` and `'Backlog'` records must be hidden.

### B. Auftragsliste / Creation List (`src/app/creation/page.tsx`)
- **Requirement**: Only show content that has actually been commissioned. Do NOT show "Planned" items.
- **Logic Change**: Refine the `commissionedKeywords` filter to strictly include:
    - `'Beauftragt'`, `'In Arbeit'`, `'Angeliefert'`, `'Review'`, `'Optimierung'`.
- **Note on Removal**: In this specific update, the `'Published'` status will **not** trigger removal from this list yet (per user request), but it will be filtered out from the Editorial Plan.

## 2. Publishing Workflow Synchronization

### A. Lifecycle Cleanup
- **Trigger**: When a user clicks "Als veröffentlicht markieren" in `AIEditorWorkspace.tsx`, the status updates to `'Published'`.
- **Automatic Sync**: 
    - The record will automatically disappear from the **Redaktions-Planung** due to the updated filter.
    - The record will reappear in the **Vorschläge** tab (since it is `Published` and thus eligible for future optimization).
    - **Exception**: The record remains visible in the **Creation Order List** for now.

## 3. Implementation Steps

1. **Update `editorial-planning.tsx`**: Modify the `data` memo to strictly filter by active workflow statuses.
2. **Update `src/app/creation/page.tsx`**: Refine the keyword filter to exclude `'Planned'` and `'Published'`.
3. **Verification**: 
    - Add a keyword from "Vorschläge" -> Verify it appears in "Redaktionsplan" but NOT in "Creation".
    - Commission content -> Verify it appears in "Creation".
    - Publish content -> Verify it disappears from both "Redaktionsplan" and "Creation".
