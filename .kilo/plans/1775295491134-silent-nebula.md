# Plan: Content-Historie & UI-Optimierung

Dieses Dokument beschreibt die Erweiterung der Content-Historie um neue Event-Typen und die Verbesserung der Timeline-UI.

## 1. Datenmodell & Events

Die Historie soll URL-basiert (gruppiert nach `Target_URL`) folgende Ereignisse erfassen:

| Event | Action_Type (Airtable) | Diff_Summary (Vorschlag) | Trigger |
| :--- | :--- | :--- | :--- |
| URL der Keyword-Map hinzugefügt | Planung | URL der Keyword-Map hinzugefügt | `POST /api/planning/keywords` |
| URL der Vorschlagsliste hinzugefügt | Planung | URL der Vorschlagsliste hinzugefügt | Statusänderung -> `Backlog` |
| URL der Redaktionsplanung hinzugefügt | Planung | URL der Redaktionsplanung hinzugefügt | Statusänderung -> `Planned` |
| Content beauftragt | Erstellung | Content beauftragt | `POST /api/n8n/trigger` |
| Content angeliefert | Erstellung | Content angeliefert | `POST /api/n8n/callback` |
| Content veröffentlicht | Optimierung | Content als veröffentlicht markiert | Statusänderung -> `Published` |

## 2. UI-Optimierung (Pop-Up / HistoryList)

Das Ziel ist eine extrem übersichtliche Timeline, die nur die relevante "Nahrungskette" an Events zeigt:

- **Filterung**: Alle Events mit dem Typ "Planung" (außer die oben definierten) werden aus der Anzeige entfernt.
- **Event-Fokus**: Jeder Eintrag zeigt nur das Event (Diff_Summary) und den Zeitpunkt (rechtsbündig).
- **Versionierung**:
  - Nur "Content angeliefert"-Events erhalten ein Versions-Badge.
  - Das erste "Content angeliefert"-Event erhält immer **V1**.
  - Jedes weitere "Content angeliefert"-Event wird hochgezählt (**V2, V3, etc.**).
  - Andere Events (Hinzugefügt, Beauftragt, Veröffentlicht) haben **keine** Versionsnummer.
- **Collapsible Content**: 
  - Nur bei "Angeliefert"-Events gibt es die Option "Content anzeigen".
  - Bei "Content beauftragt" gibt es **keine** V-Nummer und **keine** Option, Content anzuzeigen.
- **Synchronisation**: Das Pop-up auf der Seite `/history` (in `ContentHistoryTable.tsx`) nutzt dieselbe `HistoryList`-Komponente und Regeln.
